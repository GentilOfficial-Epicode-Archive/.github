# GentilOfficial Archive

This organization serves as the permanent archive for projects developed by **GentilOfficial** throughout learning paths, coursework, and personal experimentation.

The repositories are preserved as a historical record of technical growth and are no longer considered actively maintained unless explicitly stated otherwise.

## About the Archive

The archive contains projects created during the **Epicode Full Stack Developer Bootcamp**, together with additional personal exercises, prototypes, and experiments.

The bootcamp covered the following subjects:

* **Intro to Web Development**

  * HTML5
  * CSS3
  * Responsive Design
  * Git & GitHub fundamentals

* **Programming Foundations**

  * JavaScript fundamentals
  * Algorithms
  * Data structures
  * Object-Oriented Programming
  * ES6+

* **Web Design**

  * Advanced CSS
  * Bootstrap
  * UI implementation
  * Responsive layouts

* **Advanced JavaScript**

  * Modern JavaScript
  * Asynchronous programming
  * REST APIs
  * TypeScript fundamentals

* **React**

  * React
  * React Router
  * Context API
  * Redux
  * Component architecture
  * Single Page Applications

* **Express & MongoDB**

  * Node.js
  * Express
  * MongoDB
  * Mongoose
  * Authentication
  * REST API development

## Repository Status

Most repositories are archived and preserved in their original state to document the learning process.

* Dependencies may be outdated.
* The code reflects the knowledge and best practices at the time it was written.
